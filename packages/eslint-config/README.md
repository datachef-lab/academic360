# `@academic360/eslint-config`

Shared ESLint flat configs for the Academic360 monorepo.

## Configs

| Export | Use case |
| --- | --- |
| `@academic360/eslint-config/base` | Base TypeScript rules |
| `@academic360/eslint-config/node` | Node.js / backend packages |
| `@academic360/eslint-config/react` | React apps (Vite) |
| `@academic360/eslint-config/react-internal` | Internal React component libraries |
| `@academic360/eslint-config/next` | Next.js apps |
| `@academic360/eslint-config/next-js` | Alias for `next` (legacy import path) |
